# SafeSpace Student Support Platform

A modern, vibrant web app for student mental health support, featuring a forum and curated resources. Built with HTML, CSS (custom + Bootstrap), and a purple-themed UI for a calming, professional look.

## Features
- Student support forum for anonymous sharing
- Curated mental health resources
- Responsive, mobile-friendly design
- Modern purple/blue gradient UI

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in your browser to view the homepage.
3. Navigate to `forum.html` for the student forum and `resources.html` for resources.

## Folder Structure

## Customization
- Edit `css/styles.css` to change the color scheme or layout.
- Add new posts/resources by editing the respective HTML files.
